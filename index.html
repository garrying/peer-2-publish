dat is a protocol designed for syncing folders of data, even if they are large or changing constantly. dat uses a cryptographically secure register of changes to prove that the requested data version is distributed.

a group of dat clients can connect to each other to form a public or private decentralized network to exchange data between each other.

– mathias buus madsen, karissa mckelvey, maxwell ogden, code for science, 2017
